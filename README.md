# shopify theme template

Shopify theme bootstraped with Liquid / tailwindcss.

### Tech Stack

tailwind, postCSS, vanillaJS, ESBuild, preact

### Development

```bash
# dev (first time)
shopify theme dev --store=my-liquid-tailwind-template.myshopify.com
yarn dev

# dev
shopify theme dev
yarn dev

# logout
shopify auth logout
```
